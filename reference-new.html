    Cloud Volumes ONTAP 9.10.1мқҳ мғҲлЎңмҡҙ кё°лҠҘ

    04/04/2022

    Cloud Volumes ONTAP 9.10.1 м—җлҠ” м—¬лҹ¬ к°Җм§Җ мғҲлЎңмҡҙ кё°лҠҘкіј н–ҘмғҒлҗң кё°лҠҘмқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

    мөңмӢ  лІ„м „мқҳ Cloud Managerм—җлҠ” м¶”к°Җ кё°лҠҘкіј к°ңм„  мӮ¬н•ӯлҸ„ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. лҘј м°ёмЎ°н•ҳмӢӯмӢңмҳӨ "Cloud Manager лҰҙлҰ¬мҰҲ л…ёнҠё" лҘј м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

    лқјмқҙм„јмҠӨ м—…лҚ°мқҙнҠё(2022л…„ 2мӣ” 9мқј)

    Cloud Volumes ONTAPлҠ” лӢӨмқҢкіј к°ҷмқҖ мӢңмһҘ ліҖнҷ”лҘј л°ңн‘ңн–ҲмҠөлӢҲлӢӨ.

    • мқҙм ң Essentials нҢЁнӮӨм§ҖмҷҖ Professional нҢЁнӮӨм§ҖлҘј AWS л°Ҹ Google Cloud л§Ҳмј“н”Ңл ҲмқҙмҠӨм—җм„ң мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

      мқҙлҹ¬н•ң мҡ©лҹүмқ„ кё°мӨҖмңјлЎң л№„мҡ©мқ„ м§Җл¶Ҳн•  мҲҳ мһҲмңјл©°, м—°к°„ кі„м•Ҫмқ„ нҒҙлқјмҡ°л“ң кіөкёүмһҗлЎңл¶Җн„° м§Ғм ‘ кө¬л§Өн•  мҲҳ мһҲмҠөлӢҲлӢӨ. NetAppм—җм„ң м§Ғм ‘ мҡ©лҹү лқјмқҙм„јмҠӨлҘј кө¬л§Өн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

      нҒҙлқјмҡ°л“ң л§Ҳмј“н”Ңл ҲмқҙмҠӨм—җ кё°мЎҙ кө¬лҸ…мқҙ мһҲлҠ” кІҪмҡ° мқҙлҹ¬н•ң мғҲлЎңмҡҙ мҳӨнҚјл§Ғмқ„ мһҗлҸҷмңјлЎң кө¬лҸ…н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мғҲлЎңмҡҙ Cloud Volumes ONTAP мһ‘м—… нҷҳкІҪмқ„ кө¬м¶•н•  л•Ң мҡ©лҹү м¶©м „мқ„ м„ нғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

      мӢ к·ң кі к°қмқё кІҪмҡ° мғҲ мһ‘м—… нҷҳкІҪмқ„ мғқм„ұн•  л•Ң Cloud Managerм—җм„ң кө¬лҸ…мқ„ мӢ мІӯн•ҳлқјлҠ” л©”мӢңм§Җк°Җ н‘ңмӢңлҗ©лӢҲлӢӨ.

    • AWS л°Ҹ Google Cloud л§Ҳмј“н”Ңл ҲмқҙмҠӨлҘј нҶөн•ң л…ёл“ң лқјмқҙм„јмӢұмқҖ лҚ” мқҙмғҒ мӮ¬мҡ©лҗҳм§Җ м•Ҡмңјл©° мғҲ к°Җмһ…мһҗм—җкІҢ м ңкіөлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ. м—¬кё°м—җлҠ” м—°к°„ кі„м•Ҫ л°Ҹ мӢңк°„лі„ к°Җмһ…(нғҗмғү, н‘ңмӨҖ, н”„лҰ¬лҜём—„)мқҙ нҸ¬н•Ёлҗ©лӢҲлӢӨ.

      мқҙ м¶©м „ л°©лІ•мқҖ нҷңм„ұ м„ңлёҢмҠӨнҒ¬лҰҪм…ҳмқ„ ліҙмң н•ң кё°мЎҙ кі к°қм—җкІҢлҸ„ кі„мҶҚ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    "Cloud Volumes ONTAPмқҳ лқјмқҙм„јмҠӨ мҳөм…ҳм—җ лҢҖн•ҙ мһҗм„ёнһҲ м•Ңм•„ліҙмӢӯмӢңмҳӨ".

    9.10.1 GA(2022л…„ 1мӣ” 31мқј)

    Cloud Volumes ONTAP 9.10.1мқҳ GA(General Availability) лҰҙлҰ¬мҠӨк°Җ м¶ңмӢңлҗҳм—ҲмҠөлӢҲлӢӨ. GA лҰҙлҰ¬мҠӨм—җлҠ” лІ„к·ё мҲҳм •мқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

    Cloud Managerм—җм„ң 9.10.1 RC1, 9.10.0 лҳҗлҠ” 9.9.1мқҳ лІ„м „мқ„ мӢӨн–ү мӨ‘мқё кё°мЎҙ мӢңмҠӨн…ңмқ„ м—…к·ёл Ҳмқҙл“ңн•ҳлқјлҠ” л©”мӢңм§Җк°Җ н‘ңмӢңлҗ©лӢҲлӢӨ.

    лҚ” л§ҺмқҖ VM мң нҳ• м§Җмӣҗ(2021л…„ 1мӣ” 2мқј)

    Cloud Manager 3.9.14 лҰҙлҰ¬мҠӨмқҳ мқјл¶ҖлЎң Cloud Volumes ONTAPлҠ” мқҙм ң Microsoft Azureм—җм„ң 9.10.1 лҰҙлҰ¬мҠӨл¶Җн„° лӢӨмқҢ VM мң нҳ•м—җм„ң м§Җмӣҗлҗ©лӢҲлӢӨ.

    лЎң мқҙлҸҷн•©лӢҲлӢӨ "м§ҖмӣҗлҗҳлҠ” кө¬м„ұ нҺҳмқҙм§Җ" лҘј м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

    9.10.1 RC1(2021л…„ 11мӣ” 28мқј)

    Cloud Volumes ONTAP 9.10.1 м¶ңмӢң нӣ„ліҙ 1мқҖ нҳ„мһ¬ AWS, Azure л°Ҹ Google нҒҙлқјмҡ°л“ңм—җм„ң м ңкіөлҗ©лӢҲлӢӨ. мқҙ лҰҙлҰ¬мҰҲм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ мғҲлЎңмҡҙ кё°лҠҘкіј н–ҘмғҒлҗң кё°лҠҘмқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

    Microsoft Azureмқҳ HA кё°лҠҘ н–ҘмғҒ

    • Microsoft Azureм—җм„ң мӢӨн–үлҗҳлҠ” Cloud Volumes ONTAP HA мҢҚмқҳ кі„нҡҚлҗҳм§Җ м•ҠмқҖ нҺҳмқјмҳӨлІ„ мӢңк°„мқ„ мӨ„мҳҖмҠөлӢҲлӢӨ.

    • мқҙм ң NFSv4к°Җ м§Җмӣҗлҗ©лӢҲлӢӨ.

    Cloud Volumes ONTAPм—җм„ң мқҙлҹ¬н•ң н–ҘмғҒлҗң кё°лҠҘмқ„ мӮ¬мҡ©н•ҳл Өл©ҙ Azure кө¬лҸ…м—җм„ң кё°лҠҘмқ„ нҷңм„ұнҷ”н•ҙм•ј н•©лӢҲлӢӨ. "мҶҢмң мһҗ" к¶Ңн•ңмқҙ мһҲлҠ” Azure мӮ¬мҡ©мһҗлҠ” Azure CLIм—җм„ң мқҙ кё°лҠҘмқ„ нҷңм„ұнҷ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    Azure кө¬лҸ…м—җм„ң мқҙ кё°лҠҘмқ„ нҷңм„ұнҷ”н•ҙм•ј н•ҳлҠ” Cloud Volumes ONTAP 9.10.1 мӢңмҠӨн…ңмқҳ кІҪмҡ° нҒҙлқјмҡ°л“ң кҙҖлҰ¬мһҗк°Җ мқҙлҹ¬н•ң м„ёл¶Җ м •ліҙлҘј мһ‘м—… н•„мҡ” л©”мӢңм§Җм—җ н‘ңмӢңн•©лӢҲлӢӨ.

    лӢӨмқҢ мӮ¬н•ӯм—җ мң мқҳн•ҳмӢӯмӢңмҳӨ.

    • Cloud Volumes ONTAP HA мҢҚмқҳ кі к°Җмҡ©м„ұм—җлҠ” л¬ём ңк°Җ м—ҶмҠөлӢҲлӢӨ. мқҙ Azure кё°лҠҘмқҖ ONTAPмҷҖ н•Ёк»ҳ мһ‘лҸҷн•ҳм—¬ кі„нҡҚлҗҳм§Җ м•ҠмқҖ нҺҳмқјмҳӨлІ„ мқҙлІӨнҠёлЎң мқён•ҙ NFS н”„лЎңнҶ мҪңм—җ лҢҖн•ҙ нҒҙлқјмқҙм–ёнҠём—җм„ң кҙҖмёЎлҗң м• н”ҢлҰ¬мјҖмқҙм…ҳ мӨ‘лӢЁ мӢңк°„мқ„ мӨ„мһ…лӢҲлӢӨ.

    • мқҙ кё°лҠҘмқ„ мӮ¬мҡ©н•ҳлҸ„лЎқ м„Өм •н•ҳлҠ” кІғмқҖ Cloud Volumes ONTAP HA мҢҚмқҳ л¬ҙмӨ‘лӢЁ кё°лҠҘмһ…лӢҲлӢӨ.

    • Azure кө¬лҸ…м—җм„ң мқҙ кё°лҠҘмқ„ нҷңм„ұнҷ”н•ҙлҸ„ лӢӨлҘё VMм—җ л¬ём ңк°Җ л°ңмғқн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

    ліҙм•Ҳ к°•нҷ”

    ONTAP мқҙлҜём§Җ мғқм„ұ л°Ҹ Cloud Volumes ONTAP л¶ҖнҢ… л””мҠӨнҒ¬мқҳ ліҙм•Ҳ кё°лҠҘмқҙ н–ҘмғҒлҗҳм—ҲмҠөлӢҲлӢӨ.

    Keystone Flex кө¬лҸ… м§Җмӣҗ

    Cloud Manager 3.9.13 лҰҙлҰ¬мҰҲмқҳ мқјл¶ҖлЎңмҚЁ Cloud Volumes ONTAP HA pairлЎң Keystone мң м—°н•ң кө¬лҸ…мқ„ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. Keystone Flex SubscriptionмқҖ мў…лҹүм ң кө¬лҸ… кё°л°ҳ м„ңл№„мҠӨлЎң, OpEx мҶҢл№„ лӘЁлҚёмқ„ м„ нҳён•ҳлҠ” мӮ¬мҡ©мһҗм—җкІҢ мӣҗнҷңн•ң н•ҳмқҙлёҢлҰ¬л“ң нҒҙлқјмҡ°л“ң кІҪн—ҳмқ„ м ңкіөн•ҳм—¬ мһҗліё м§Җм¶ң лҳҗлҠ” мһ„лҢҖлҘј м§Җмӣҗн•©лӢҲлӢӨ.

    Keystone мң м—°н•ң к°Җмһ…мқҖ Cloud Managerм—җм„ң кө¬м¶•н•  мҲҳ мһҲлҠ” лӘЁл“  мғҲлЎңмҡҙ лІ„м „мқҳ Cloud Volumes ONTAPм—җм„ң м§Җмӣҗлҗ©лӢҲлӢӨ.

    м•”нҳёнҷ” нӮӨ кҙҖлҰ¬

    ONTAP 9.10.1 лҰҙлҰ¬мҠӨм—җлҠ” ONTAP м•”нҳёнҷ” нӮӨлҘј ліҙнҳён•ҳкё° мң„н•ҙ AKV(Azure Key Vault) лҳҗлҠ” Google нҒҙлқјмҡ°л“ң нӮӨ кҙҖлҰ¬ м„ңл№„мҠӨлҘј мӮ¬мҡ©н•  мҲҳ мһҲлҠ” н–ҘмғҒлҗң кё°лҠҘмқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

    "Cloud Volumes ONTAP м•”нҳёнҷ” м§Җмӣҗм—җ лҢҖн•ҙ мһҗм„ёнһҲ м•Ңм•„ліҙмӢӯмӢңмҳӨ".

    Cloud Manager Connectorмқҳ н•„мҲҳ лІ„м „мһ…лӢҲлӢӨ

    мғҲлЎңмҡҙ Cloud Volumes ONTAP 9.10.1 мӢңмҠӨн…ңмқ„ л°°нҸ¬н•ҳкі  кё°мЎҙ мӢңмҠӨн…ңмқ„ 9.10.1лЎң м—…к·ёл Ҳмқҙл“ңн•ҳл Өл©ҙ Cloud Manager Connectorк°Җ лІ„м „ 3.9.13 мқҙмғҒмқ„ мӢӨн–үн•ҙм•ј н•©лӢҲлӢӨ.

    Connectorмқҳ мһҗлҸҷ м—…к·ёл Ҳмқҙл“ңлҠ” кё°ліём ҒмңјлЎң нҷңм„ұнҷ”лҗҳм–ҙ мһҲмңјлҜҖлЎң мөңмӢ  лІ„м „мқ„ мӢӨн–үн•ҙм•ј н•©лӢҲлӢӨ.

    л…ёнҠё м—…к·ёл Ҳмқҙл“ң

    • Cloud Volumes ONTAP м—…к·ёл Ҳмқҙл“ңлҠ” Cloud Managerм—җм„ң мҷ„лЈҢн•ҙм•ј н•©лӢҲлӢӨ. System Manager лҳҗлҠ” CLIлҘј мӮ¬мҡ©н•ҳм—¬ Cloud Volumes ONTAPлҘј м—…к·ёл Ҳмқҙл“ңн•ҙм„ңлҠ” м•Ҳ лҗ©лӢҲлӢӨ. мқҙл ҮкІҢ н•ҳл©ҙ мӢңмҠӨн…ң м•Ҳм •м„ұм—җ мҳҒн–Ҙмқ„ мӨ„ мҲҳ мһҲмҠөлӢҲлӢӨ.
